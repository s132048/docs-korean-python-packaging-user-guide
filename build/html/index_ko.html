파이썬 패키징 사용자 가이드¶

이 파이썬 패키징 사용자 가이드는 현대적인 툴로 파이썬 패키지를 배포하고 설치할 수 있게 도움을 주는 튜토리얼과 레퍼런스 모음집이다.

이 가이드는 Python Packaging Authority에 의해 Github 에서 관리되고 있다. 우리는 어떤 식의 기여와 피드백도 환영한다. 😊

Note

legacy PyPI에서 pypi.org로 옮기는 것에 대한 가이드를 찾고 있다면 PyPI.org로 마이그레이션하기를 참고하라.

시작하기¶

파이썬 패키징 생태계에서 일하는 것에 대한 개념과 핵심적인 툴은 튜토리얼 섹션에서 다루어지고 있다:

  • 패키지 설치에 대한 방법은 패키지 설치하기를 참고하라.
  • 프로젝트를 패키징하고 배포하는 방법은 프로젝트 패키징과 배포를 참고하라.

더 알아보기¶

이 가이드는 튜토리얼 이외에 여러가지 다른 자료들을 담고 있다:

  • 리눅스 패키지 매니저로 pip/setuptools/wheel 설치하기, 바이너리 확장 패키징 등에 대한 설명이 담긴 가이드 섹션
  • Deploying Python applications, pip vs easy_install 등의 심층적인 주제를 다룬 토론 섹션
  • 상호 운용 사양 패키징에 관한 PyPA 설명서 섹션

그리고 Python Packaging Authority 멤버에 의해 관리되는 다른 프로젝트 리스트가 존재한다.
